스마트 와이어
------
실시간 기계 상태 알림 프로그램

### 프로젝트 소개
- 제조 공장에 1년 간 일하면서 직접 다뤘던 기계의 문제점을 해결하고자 시작한 프로젝트 입니다.
- 모든 기계의 상태를 한 눈에 핸드폰, 컴퓨터로 확인 할 수 있습니다. 
- 기계가 멈추면 핸드폰으로 알림을 보내줍니다.
- 이 때 추가로 팀뷰어, 애니데스크 등 무료 원격 제어 프로그램을 사용하여 기계를 다시 작동시킬 수도 있습니다.
